## Usage

```md

```

```md

```

## ➕ Add your theme

open an [issue](https://github.com/fxhxyz4/readme-quotes/issues?q=state%3Aopen%20label%3Atheme) with the `theme` label and paste this:

```json
{
  "name": "theme name",
  "bg": "#hex",
  "text": "#hex",
  "author": "#hex"
}
```
